Font & Text Related Properties : 1.font-size : size of font Ex: %px. 2.font-weight : thickness Ex: normal,bold,lighter. 3.font-style : style of font Ex: normal,italic,oblique. 4.font-family : font family name Ex:Arial,sans-sarif; Arial,Helvetica,sans-sarif; 1.line-height : space b/w lines of text. 2.letter-spacing : space b/w letters. 3.word-spacing : space b/w words. 4.text-transorm : Capitalization Ex:(upper,lowerCase,Capitalize). 5.text-align : Horizental Alignment Ex:left,right,center. 6.text-indent : Indentation of first line in a text block. 7.text-decoration : Adds/Remove text decorations. Ex: underline,overline,line-through. 8.text-shadow : Adds a shadow effect to text Ex: 'text-shadow:2px 2px 5px gray;' Cascading : Applying Styles based on priority. Inline styles > Internal styles > External styles. ID selector > Class selector > Element Name selector. !important means important having highest priority in CSS. 1.Universal selector(*) : selects all elements on page. *{ color:red; font-style:italic; } 2.Group selector : selects multiple elements at once. h1,h2,h3{ font-family:Arial,sans-serif; } 3.Descendant selector : select elements that are descendant of a specific element. div p{ color:green; } the styles are applied on div element paragraph. div p span{ color:red; font-style:italic; } the styles are applied div element paragraph span. Box Model: Box Model Contains the 4-Elements: 1.Content : Actual content of element like text,image. 2.Border : Border Around Box. 3.Padding : Space b/w Content & it's Border.(inside) 4.Margin : Space b/w Box & Box.(outside) Padding Properties : padding-top: 5px; padding-right: 10px; padding-bottom: 15px; padding-left: 20px; padding: 5px 10px 15px 20px; (top,right,bottom,left). padding: 5px 10px; (top/bottom:5px, right/left:10px). padding:10px; (all sides have 10 pixels).
