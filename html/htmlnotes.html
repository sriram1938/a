			Anchor tag 

Used to create hyperlinks that navigating to another webpage.
name of link
				
	href (Hyperlink Reference): specifies the URL/path to navigate to.
	target='defines where to open the linked document.'
		some targets:	'_self'=Opens in the same tab.
				'_blank'=Opens in a new tab.
				'_parent'=Opens in the parent frame.
				'_top'=Opens in the full body of the window.
	rel='the relationship b/w the current and linked document.'
	title='Provides additional information about the link as a tooltip.'
	type='specifies the type of the linked resource (less used).'
	accesskey='provides a shortcut key to focus/activate the link.'
